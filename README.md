# Waste_sorting_ABM_Zapopan
This repository contains the current version of the ABM used in the article "A SEM-based ABM Simulation for Environmental studies: The case of Household Waste Sorting in Developing Countries." 


Installation
Clone this reposetory
Download GAMA (compatible with GAMA 1.9.3) 
Run GAMA,
Choose a new Workspace (this is a temporay folder used for computation)
right click on User Models->Import->GAMA Project..
Select appropiate folder that you have clone
